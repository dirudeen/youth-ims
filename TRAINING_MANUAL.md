## 1. System Overview

### What is the Youth IMS?

The Youth Information Management System (Youth IMS) is a comprehensive web-based platform developed for the Ministry of Youth and Sports of The Gambia. The system is designed to centralize, manage, and analyze data related to youth programs, demographics, and initiatives across the country.

### Key Objectives

- **Centralized Data Management**: Store all youth-related data in one secure location
- **Real-time Analytics**: Generate insights through interactive dashboards and charts
- **Program Monitoring**: Track the progress and impact of various youth programs
- **Evidence-based Decision Making**: Support policy development with accurate data
- **Stakeholder Collaboration**: Enable multiple departments and organizations to access and contribute data

### System Modules

The Youth IMS consists of the following main modules:

| Module | Description |
|--------|-------------|
| **Dashboard** | Overview of key statistics and trends |
| **Youth Population** | Demographics data by region and age group |
| **Youth with Disabilities** | Data on youth with various disability types |
| **Youth without Disabilities** | Age group distribution data |
| **Human Trafficking** | Cases and interventions tracking |
| **Youth Migration** | Migration patterns and statistics |
| **NYC (National Youth Council)** | Activities and beneficiaries data |
| **NSC (National Sports Council)** | Sports programs and participants |
| **NYSS (National Youth Service Scheme)** | Service scheme programs and graduates |
| **PIA (President's International Award)** | Award program participants |
| **NEDI (National Enterprise Development Initiative)** | Entrepreneurship programs |
| **Indicator Data** | RF-NDP performance indicators |
| **Sports Financing** | Sports federation funding data |

## 2. Getting Started

### System Requirements

- Modern web browser (Chrome, Firefox, Edge, or Safari)
- Stable internet connection
- User account credentials provided by the system administrator

### Logging In

1. Open your web browser and navigate to the Youth IMS URL
2. Enter your **email address** in the email field
3. Enter your **password** in the password field
4. Click the **"Sign In"** button
5. Upon successful login, you will be redirected to the Dashboard

### First-time Login

If this is your first time logging in:
- Use the temporary password provided by your administrator
- You may be prompted to change your password
- Update your profile information if required

### Logging Out

1. Click on your **profile icon** in the top-right corner of the header
2. Select **"Sign Out"** from the dropdown menu
3. You will be redirected to the login page

## 3. Navigating the System

### Main Navigation Elements

#### Sidebar (Left Panel)
The sidebar contains links to all system modules:
- **Dashboard** - Main overview page
- **Youth Population** - Demographic data
- **Youth with Disabilities** - Disability statistics
- **Youth without Disabilities** - Age distribution
- **Human Trafficking** - Case management
- **Youth Migration** - Migration data
- **NYC** - National Youth Council activities
- **NSC** - National Sports Council data
- **NYSS** - Youth Service Scheme
- **PIA** - President's International Award
- **NEDI** - Enterprise development
- **Indicator Data** - Performance indicators
- **Sports Financing** - Funding data
- **Activity Log** - System activity history
- **User Management** - User administration (Admin only)

#### Header (Top Bar)
- **Search Bar**: Use Ctrl+K (or Cmd+K on Mac) to quickly search across all modules
- **User Profile**: Access your profile settings and logout option

### Using the Global Search

1. Press **Ctrl+K** (Windows) or **Cmd+K** (Mac) to open the search dialog
2. Type your search query (e.g., "youth population", "NSC", "migration")
3. Select from the search results to navigate directly to that page

## 5. Data Management Modules

### Viewing Data

Each module displays data in a table format with the following features:
- **Column Headers**: Click to sort data ascending/descending
- **Pagination**: Navigate between pages of records
- **Search/Filter**: Use filters to find specific records

### Adding New Records

1. Navigate to the desired module (e.g., NYC Activities)
2. Click the **"Add New"** or **"+"** button
3. Fill in all required fields in the form
4. Click **"Save"** or **"Submit"** to add the record

### Editing Records

1. Locate the record you want to edit in the table
2. Click the **Edit** (pencil icon) button in the Actions column
3. Modify the necessary fields
4. Click **"Save"** to update the record

### Deleting Records

1. Locate the record you want to delete
2. Click the **Delete** (trash icon) button in the Actions column
3. Confirm the deletion when prompted

> **Note**: Deletion is permanent and cannot be undone. Only users with appropriate permissions can delete records.

## 6. User Roles and Permissions

### Role Types

| Role | Permissions |
|------|-------------|
| **Super Admin** | Full access to all modules, user management, and system settings |
| **Admin** | Can view, add, edit, and delete data across all modules |
| **Editor** | Can view, add, and edit data but cannot delete |
| **Viewer** | Read-only access to view data and reports |

### Permission Details

- **View**: Can see data and generate reports
- **Add**: Can create new records
- **Edit**: Can modify existing records
- **Delete**: Can remove records from the system
- **Manage Users**: Can create, edit, and deactivate user accounts

## 7. Common Tasks

### Task 1: View Youth Population by Region

1. Click **"Youth Population"** in the sidebar
2. Navigate to the **"Analytics"** tab
3. Use the region filter to select a specific region
4. View the population breakdown in the chart

### Task 2: Add a New NYC Activity

1. Click **"NYC"** in the sidebar
2. Click the **"Add Activity"** button
3. Fill in the activity details:
   - Activity name
   - Date/Year
   - Region/LGA
   - Number of beneficiaries (male/female)
   - Description
4. Click **"Save"**

### Task 3: Generate an Indicator Report

1. Click **"Indicator Data"** in the sidebar
2. Use filters to select:
   - Organization (NYC, NYSS, NSC, etc.)
   - Year
   - Indicator type
3. View the filtered data and statistics
4. Export or print as needed

### Task 4: Create a New User (Admin Only)

1. Click **"User Management"** in the sidebar
2. Click **"Add User"** button
3. Enter user details:
   - Full Name
   - Email Address
   - Password
   - Role (Super Admin, Admin, Editor, Viewer)
   - Organization/Department
4. Click **"Create User"**

### Task 5: Check System Activity

1. Click **"Activity Log"** in the sidebar
2. View recent system activities
3. Use filters to search by:
   - User
   - Action type
   - Date range

## 8. Tips and Best Practices

### Data Entry

- **Be Accurate**: Double-check all data before saving
- **Be Complete**: Fill in all required fields
- **Be Consistent**: Use consistent naming conventions and formats
- **Verify Sources**: Ensure data comes from reliable sources

### Navigation

- Use the **global search** (Ctrl+K) for quick navigation
- **Bookmark** frequently used pages in your browser
- Use browser **back/forward** buttons to navigate between pages

### Security

- **Never share** your login credentials
- **Log out** when leaving your workstation
- **Report** any suspicious activity to the administrator
- Use a **strong password** with a mix of letters, numbers, and symbols

### Performance

- **Clear browser cache** periodically for optimal performance
- Use a **stable internet connection**
- **Avoid** opening too many browser tabs simultaneously

## 9. Troubleshooting

### Common Issues and Solutions

#### Issue: Cannot Log In

**Possible Causes:**
- Incorrect email or password
- Account has been deactivated
- Browser cookies/cache issues

**Solutions:**
1. Verify your email and password are correct
2. Contact the administrator to check your account status
3. Clear your browser cache and cookies, then try again

#### Issue: Data Not Loading

**Possible Causes:**
- Slow internet connection
- Server maintenance
- Browser compatibility issues

**Solutions:**
1. Check your internet connection
2. Refresh the page (F5 or Ctrl+R)
3. Try a different browser
4. Contact the administrator if the issue persists

#### Issue: Cannot Save Data

**Possible Causes:**
- Missing required fields
- Permission restrictions
- Session timeout

**Solutions:**
1. Ensure all required fields (marked with *) are filled
2. Check if you have permission to add/edit data
3. Log out and log back in, then try again

#### Issue: Page Not Displaying Correctly

**Possible Causes:**
- Browser zoom level
- Outdated browser
- Screen resolution issues

**Solutions:**
1. Reset browser zoom to 100% (Ctrl+0)
2. Update your browser to the latest version
3. Try adjusting your screen resolution

### Getting Help

If you encounter issues not covered in this manual:

1. Contact your **System Administrator**
2. Submit a **support ticket** through the appropriate channels
3. Document the issue with:
   - Screenshot of the error
   - Steps to reproduce the problem
   - Date and time of occurrence
